# The Last Breath

## 🎮 About

The Last Breath is a 2D survival horror game developed with Unity and C#.

The player must survive in a dangerous post-apocalyptic world by exploring abandoned locations, collecting resources, avoiding enemies, and completing missions.

---

## 🚀 Features

- 2D Survival Horror
- Player Movement
- Enemy AI
- Health System
- Inventory System
- Item Collection
- Weapons
- Sound Effects
- Game UI
- Save System (Planned)
